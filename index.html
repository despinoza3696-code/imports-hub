📂 Cómo actualizar los datos

El dashboard tiene los datos incrustados en el archivo HTML. Para actualizarlo con un nuevo Excel:

  1. Abre una conversación con Claude (claude.ai)
  2. Sube tu nuevo archivo .xlsx con el mismo formato
  3. Escríbele: "Actualiza mi dashboard de importaciones con este Excel"
  4. Claude generará un nuevo hub_final.html actualizado
  5. Comparte ese nuevo archivo con gerencia
